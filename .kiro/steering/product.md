# Product Overview

**SpiderX Interface** is a Next.js web application that serves as the frontend for the SpiderX platform — a productivity and AI-assisted workspace tool.

## Core Features

- **Todo / Task Management** — full CRUD task list with swipe gestures, subtask generation, thread filtering, sync, and calendar views
- **AI Inbox** — AI-generated suggestions with confidence scoring, evidence viewer, quality flags, and bulk actions
- **Google Chat Integration** — whitelist management, space browsing, AI extraction settings, and activity logs
- **Mind Map Editor** — interactive mind map creation using both `simple-mind-map` and `mind-elixir` libraries, with AI suggestions, export/import, and collaborative editing via Yjs/WebRTC
- **Marketing / Landing Pages** — versioned landing pages (v1, v2, v3) with pricing, hero sections, and visual effects
- **Auth** — sign-in / sign-up flows with JWT-based session management and route guards

## Target Users

Developers and teams who need an AI-augmented task and knowledge management interface integrated with communication tools like Google Chat.
